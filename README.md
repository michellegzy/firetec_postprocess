# firetec_postprocess

## Description
This repo contains various python files for post processing firetec runs. Currently, firetec_binary_postprocess.py is the main postprocessing script which is used for the firetec dev code. it computes various classical fire functionalities and creates csvs for further plotting/processing.for_tran_outs_1.4.1.py was created for firetec-1.4.1 and can be used similarly to firetec_binary_postprocess for the 1.4.1 code. qplot.py plots various markers for fire based ont he csvs stored from  firetec_binary_postprocess.py. minmax is a WIP code used to compute min maxes of XV array values.
